# The Parlee Data Set
* I used the ParlEE V2 dataset. You need to download it from [here](https://doi.org/10.7910/DVN/VOPK0E),even if you downloaded the TF-IDF matrices, this is needed for another part of the analysis
* ParLEE plenary speeches V2 dataset: Sylvester, C. (Creator), Greene, Z. (Creator), Ershova, A. (Contributor), Khokhlova, A. (Contributor), Yordanova, N. (Creator) (21 Feb 2023). ParlEE plenary speeches V2 data set: Annotated full-text of 15.1 million sentence-level plenary speeches of six EU legislative chambers. Harvard Datavers
